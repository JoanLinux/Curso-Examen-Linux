# Manual del Alumno - CursoLinux

## 1) Objetivo
Este manual explica como usar la plataforma de CursoLinux para tomar clase y responder el examen practico.

## 2) Antes de iniciar
- Ten a la mano tu **nombre completo** y tu **correo**.
- Usa siempre el **mismo correo** en cada intento para conservar tu historial.
- Recomendado: usa tambien el mismo nombre para facilitar seguimiento del maestro.

## 3) Ingreso a la plataforma
1. Abre la liga del curso.
2. Da clic en **Entrar al Curso**.
3. Captura:
   - Nombre completo
   - Correo
4. Presiona **Iniciar examen**.

## 4) Estructura del examen
Veras preguntas de tres tipos:
- **Opcion multiple**: selecciona la respuesta correcta.
- **Analisis visual**: da clic en la zona correcta de la imagen (`top/htop`).
- **Mini-shell**: escribe comandos como en terminal Linux.

## 5) Navegacion
- **Responder y continuar**: envia respuesta y avanza.
- **Pregunta anterior**: te permite regresar.
- Menu lateral con numeros:
  - Puedes volver a preguntas previas.
  - No puedes saltar hacia adelante a preguntas no liberadas.

## 6) Temporizador
- El examen muestra un cronometro.
- No limita el tiempo, pero registra cuanto tardas.
- Al terminar, el tiempo se congela automaticamente.

## 7) Reglas para mini-shell
- Escribe comandos completos (ejemplo: `grep -i error /var/log/syslog | tail -n 20`).
- Puedes usar prueba y error.
- Si te equivocas, vuelve a intentar.
- El entorno es simulado para practica, no afecta servidores reales.

## 8) Buenas practicas para responder
- Lee completa la consigna antes de escribir.
- Si es comando, cuida ruta, opciones y orden.
- Si es imagen, identifica primero el problema (CPU, memoria, swap, zombie, etc.).
- No adivines rapido: prioriza precision.

## 9) Finalizacion
- En la ultima pregunta se calcula tu resultado.
- El sistema guarda:
  - Puntaje
  - Tiempo total
  - Tiempos por pregunta
  - Respuestas dadas

## 10) Soporte rapido
Si algo no carga o se queda en "Cargando...":
1. Recarga la pagina.
2. Verifica internet.
3. Vuelve a entrar con el **mismo correo**.
4. Si persiste, avisa al maestro.
